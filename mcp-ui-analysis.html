Agentic Protocol

The MCP-UI Protocol Architecture

Shifting from narrative interfaces to autonomous execution environments.

Composable Fragments

Self-contained UI islands exported via the Model Context Protocol, enabling brand-native experiences within third-party hosts.

Protocol Primitives

Leveraging Resources to mount UI definitions and Tool Calls to synchronize state between the user and the LLM Orchestrator.

System Identity

Preserving design language integrity through sandboxed styling, ensuring the bank or store feels "real" inside the chat.

Islands Architecture

In this ecosystem, the LLM is the Conductor. It identifies the user's intent and mounts the appropriate functional island rather than simply explaining the data.

"The assistant acts as a composer of fragments, not a narrator of data."

Trust & Liability Gaps

Why institutional silos remain an existential barrier to pure MCP-UI adoption.

Trust Proxy Vulnerability

Phishing vectors shift to the host. If a user trusts ChatGPT, they implicitly trust the malicious bank widget rendered within it. This "inherited trust" bypasses traditional security intuition.

Economic Erosion

Banks are Destination Platforms. MCP-UI reduces them to Headless Utilities. Once a bank is just a button in a chat, their brand moat and cross-selling opportunities vanish.
