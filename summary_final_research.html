Executive Summary

The First New UI Paradigm
in 60 Years

A shift from Command-Based ("Do this, click that") to Intent-Based ("Here is the result I want") interaction.

The Challenge

The Affordance Gap

AI creates a new usability crisis: users do not know what they can ask for.

37%

Faster Execution

Speed gain for info retrieval using GenUI vs click paths.

+19%

Error Risk

Increase when users over-rely on AI without verification.

2s

Threshold

If prompting takes longer than clicking, users revert to GUI.

Research Insights

5 Core Shifts

01

Malleable Software

UI generated at runtime based on intent. The interface is the response.

02

Sandwich Workflow

Human sets context → AI generates → Human verifies. Users reject full automation.

03

Inverse IA

Start with the destination (via intent), system reconstructs the navigation path.

04

Latency Trade-off

If AI >1s latency, it must provide >10x value. AI cannot replace nav until instant.

05

Contextual Chips

Dynamic buttons that suggest intents based on context. Bridges Recognition (GUI) and Recall (AI).

The Verdict

Hybrid Interfaces Win

Neither pure GUI nor pure AI. The future is intelligent augmentation of existing applications with context-aware, intent-based interactions.
